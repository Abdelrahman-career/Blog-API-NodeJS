## Description
Blog-API-NodeJS is a simple project for managing blog posts using Node.js and Express.

## Project Status
This is a demo / non-final project — for showcasing the idea only.

## How to Run
1. Copy `.env.example` to `.env` and fill in the values.
2. Run `npm install`
3. Run `npm run dev`
